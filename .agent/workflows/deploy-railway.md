// turbo-all
1. Add all changes to git
`git add .`

2. Commit changes with a descriptive message
`git commit -m "🚀 Auto-deploy: Updated project components"`

3. Push to the main branch
`git push origin main`

4. Inform the user to check Railway dashboard for deployment status.
